\section{Configuration}
\defaultauthor

\begin{itemize}
  \item Backend: OneMap credentials, LTA API key, Gemini/Vertex settings, Supabase service role, OpenWeather key và frontend CORS URL.
  \item Frontend: API base URL và Supabase public configuration.
  \item Database: chạy migrations trong \code{supabase/migrations}.
  \item Health check: backend expose endpoint health cho platform monitoring.
\end{itemize}

Không đưa secret vào source control hoặc báo cáo.
